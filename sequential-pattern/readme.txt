How to execute?

1. check you have installed R
2. find main.R, change configuration: support and workingDir
   (input file "seqdata.dat" must in the same directory with main.R)
3. > source(main.R)
4. output file is "seqoutput.txt"

Note:
"output.txt" is temp data transformed from "seqdata.dat"
Format: sequenceId, eventTime, itemset size, itemset

Detail information: http://en.wikibooks.org/wiki/Data_Mining_Algorithms_In_R/Sequence_Mining/SPADE
